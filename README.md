# SiLa Challenge

## Description 
Ce repository contient mes solutions du challenge de coding de 30 jours.
Chaque jour, je complete une nouvelle tache et documente mon approche, ma solution et mon code.

## Structure
- Les fichiers 'JourX/' contiennent les solutions quotidiennes.
- Chaque fichier contient:
 - Un fichier README expliquant le problème et ma solution
 - Les fichiers de code source implémentant la solution.

## Règles 
- La deadline quotidienne: 22h (du jour prochain).
- Manquer à une fois au challenge implique une élimination directe.
- Le premier à atteindre les 15 points peut ajouter une nouvelle règle.
